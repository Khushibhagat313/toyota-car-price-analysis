# Toyota Car Price Exploratory Data Analysis (EDA)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Toyota Corolla dataset to understand the factors that influence car prices.

The analysis focuses on exploring relationships between different vehicle features such as age, mileage, fuel type, engine specifications, and other attributes that affect the resale value of Toyota cars.

The project uses Python data analysis libraries to clean, analyze, and visualize the dataset.

---

## Dataset
The dataset used in this project contains information about Toyota Corolla vehicles including:

- Price
- Age
- KM (Mileage)
- Fuel Type
- Horsepower (HP)
- Engine specifications
- Other vehicle attributes

The dataset helps analyze how different features influence the price of used cars.

## Project Workflow

The following steps were performed during the analysis:

1. Importing required libraries
2. Loading and inspecting the dataset
3. Data cleaning and preprocessing
4. Exploratory Data Analysis (EDA)
5. Visualizing distributions using histograms
6. Identifying outliers using boxplots
7. Analyzing categorical variables using bar graphs
8. Exploring relationships between variables using scatter plots
9. Creating correlation matrix
10. Visualizing correlations using heatmap
11. Handling categorical data
12. Preparing dataset for machine learning (train-test split)

---

## Key Insights

- Vehicle age has a strong influence on car price.
- Cars with higher mileage tend to have lower resale values.
- Certain fuel types may impact pricing differently.
- Several vehicle attributes show correlations with price.

These insights help understand the key factors affecting the resale value of Toyota cars.
